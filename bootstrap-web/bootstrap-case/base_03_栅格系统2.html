从堆叠到水平排列

华夏经纬网11月4日讯：据台湾媒体报道，台湾地区前领导人马英九4日上午参加东吴大学举行的“南海仲裁案暨国际法研讨会”，他强调，太平岛符合国际法岛屿之地位而非礁石，并在现场提供来自太平岛的饮水供中外学者饮用。 据报道，台湾东吴大学新闻稿称，马英九在说明南海岛屿的国际法地位时特别强调，太平岛符合国际法岛屿之地位而非礁石，并提供来自太平岛的饮水供中外学者饮用，以证明该岛之国际法地位。 他认为，太平岛应享有200海浬专属经济区与大陆礁层；与会学者认为，该项仲裁判断完全属于政治动机，违反仲裁员中立与善意解释条约原则。
